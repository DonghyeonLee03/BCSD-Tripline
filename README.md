# BCSD-Tripline
[BCSD] FE 회고 프로젝트로 국문 관광정보 api와 카카오맵 api를 활용한 여행 관광지 추천 사이트입니다.
## 프로젝트 소개
"**Tripline**"은 **여행**(**Trip**)과 **선**(**line**)이 결합한 이름으로 이 사이트에서 만들어낸 여행 경로는 선으로 시각화가 이루어집니다.

## 기술 스택
- React + TypeScript
- TanStack Query
- Zustand

## 주요 기능
- 위치 기반 관광지 조회
- 카테고리별 관광지 조회
- 관광지 선택 및 지도 마커 표시
- 관광 루트 시각화
